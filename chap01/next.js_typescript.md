# 1.1 Next.js와 타입스크립트   

## Next.js   
- 오픈 소스 웹 애플리케이션 프레임워크   
- 리액트를 기반으로 구현 및 개발   
- 서버 사이드 렌더링, 정적 웹사이트 생성 등   



## 기존 리액트 애플리케이션 구성 vs Next.js 구성   
기존 - 리액트가 웹 프론트엔드 렌더링에 특화된 라이브러리이기 때문 모든 콘텐츠를 클라이언트 사이드인 브라우저에서 렌더링   
Next.js - 서버 사이드에서 렌더링되는 애플리케이션으로 확장 가능
